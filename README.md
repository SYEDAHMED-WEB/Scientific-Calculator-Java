# 🔬 Scientific Calculator

A simple and interactive **Scientific Calculator** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic as well as scientific calculations through an easy-to-use interface.

## 🚀 Live Demo
Live Demo: https://syedahmed-web.github.io/Scientific-Calculator-Java/

## ✨ Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Modulus Calculation
- 🔢 Power Calculation
- √ Square Root
- 📐 Trigonometric Functions (Sin, Cos, Tan)
- 🧹 Clear Screen Function
- ⌫ Delete Last Character
- 📱 Responsive Design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Visual Studio Code
- GitHub Pages (for deployment)
